# Contributing

이 저장소는 포트폴리오용 프론트엔드 프로토타입입니다. 변경할 때는 화면 완성도와 모바일 사용성을 우선합니다.

## 작업 기준

- 모바일 뷰에서 먼저 확인합니다.
- 색상과 간격은 `src/styles.css`의 CSS 변수와 기존 패턴을 우선 사용합니다.
- 기능을 추가할 때는 `src/app.js`의 mock data 흐름을 깨지 않게 유지합니다.

## 검증

```bash
npm run check
```

## 커밋 메시지

```text
feat: 새 기능
fix: 버그 수정
docs: 문서 수정
style: UI/CSS 수정
chore: 설정 변경
```
