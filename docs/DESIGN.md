# Design Notes

## Mood

TMI Nearby는 밝은 커뮤니티 앱보다 어두운 모바일 피드에 가깝습니다. 주변 사람들의 이야기를 훔쳐보는 듯한 익명성과, 실시간으로 무언가 떠오르는 느낌을 같이 가져가려고 했습니다.

## Visual principles

| 원칙 | 설명 |
| --- | --- |
| 어두운 배경 | 밤이나 실내에서 열어도 부담 없는 화면 |
| 라임 포인트 | 반경, 실시간 상태, 작성 버튼처럼 중요한 액션만 강조 |
| 카드 스택 | 주변 이야기를 하나씩 넘겨보는 탐색 방식 |
| 둥근 모서리 | iOS 앱 같은 부드러운 모바일 인상 |
| 짧은 텍스트 | 사용자가 오래 읽기보다 빠르게 반응하도록 설계 |

## Main screens

| 화면 | 설명 |
| --- | --- |
| Feed | 근처 TMI와 투표를 카드 스택으로 보여주고, 드래그 중 LIKE/NOPE 스탬프로 방향을 피드백 |
| Card menu | 카드의 `⋯` 버튼에서 숨기기, 신고하기, 차단하기를 짧은 메뉴로 제공 |
| Vote card | 선택한 투표 옵션은 하이라이트하고, 한 번 투표한 카드는 재선택을 막음 |
| Detail sheet | 반응과 댓글을 확인하고 짧은 댓글을 작성하는 하단 시트 |
| Compose sheet | TMI 또는 투표를 작성하는 화면, 경고 배너로 쿨다운/필터링 상태 안내 |
| Notifications | 주변 반응과 댓글 알림을 보여주고, 필요 없는 알림은 `×`로 닫음 |
| Profile | 익명 프로필과 내 활동 |

## Color

| Token | Value | Usage |
| --- | --- | --- |
| Background | `#050507` | 전체 배경 |
| Surface | `rgba(255,255,255,0.055)` | 카드, 리스트, 모달 |
| Accent | `#ccff00` | 실시간 상태, 좋아요, FAB |
| Danger | `#ff5c5c` | NOPE 스탬프, 신고성 액션 후보 |
| Warning surface | `rgba(255,92,92,0.14)` | 작성 경고 배너 |
| Warning border | `rgba(255,92,92,0.4)` | 작성 경고 배너 테두리 |
| Toast surface | `rgba(12,12,15,0.96)` | 숨김, 신고, 차단 안내 토스트 |
| Text | `#f7f7f7` | 주요 텍스트 |

## Interaction tone

- 사용자가 깊게 고민하지 않고 넘길 수 있어야 합니다.
- 댓글은 전체 화면 전환보다 하단 시트가 어울립니다.
- 작성은 한 번에 짧게 끝나야 합니다.
- 실명보다 익명 캐릭터가 서비스 분위기에 맞습니다.
- 숨기기, 신고, 차단처럼 무거운 액션은 메뉴 안에 두고, 결과는 토스트로 짧게 확인시킵니다.
- 투표 선택과 알림 닫기는 즉시 반응하되 화면 흐름을 끊지 않아야 합니다.
