# Design Notes

## Mood

TMI Nearby는 밝은 커뮤니티 앱보다 어두운 모바일 피드에 가깝습니다. 주변 사람들의 이야기를 훔쳐보는 듯한 익명성과, 실시간으로 무언가 떠오르는 느낌을 같이 가져가려고 했습니다.

## Visual principles

| 원칙 | 설명 |
| --- | --- |
| 어두운 배경 | 밤이나 실내에서 열어도 부담 없는 화면 |
| 라임 포인트 | 반경, 실시간 상태, 작성 버튼처럼 중요한 액션만 강조 |
| 카드 스택 | 주변 이야기를 하나씩 넘겨보는 탐색 방식 |
| 둥근 모서리 | iOS 앱 같은 부드러운 모바일 인상 |
| 짧은 텍스트 | 사용자가 오래 읽기보다 빠르게 반응하도록 설계 |

## Main screens

| 화면 | 설명 |
| --- | --- |
| Feed | 근처 TMI와 투표를 카드 스택으로 보여주는 메인 화면 |
| Detail sheet | 반응과 댓글을 확인하는 하단 시트 |
| Compose sheet | TMI 또는 투표를 작성하는 화면 |
| Notifications | 주변 반응과 댓글 알림 |
| Profile | 익명 프로필과 내 활동 |

## Color

| Token | Value | Usage |
| --- | --- | --- |
| Background | `#050507` | 전체 배경 |
| Surface | `rgba(255,255,255,0.055)` | 카드, 리스트, 모달 |
| Accent | `#ccff00` | 실시간 상태, 좋아요, FAB |
| Danger | `#ff5c5c` | 넘기기, 신고성 액션 후보 |
| Text | `#f7f7f7` | 주요 텍스트 |

## Interaction tone

- 사용자가 깊게 고민하지 않고 넘길 수 있어야 합니다.
- 댓글은 전체 화면 전환보다 하단 시트가 어울립니다.
- 작성은 한 번에 짧게 끝나야 합니다.
- 실명보다 익명 캐릭터가 서비스 분위기에 맞습니다.
