# Product Notes

## 문제 정의

동네나 같은 공간에 있는 사람들과 가볍게 말을 섞고 싶지만, 실명 기반 커뮤니티는 부담스럽고 일반 SNS는 너무 넓습니다. TMI Nearby는 아주 짧은 반경 안에서만 보이는 익명 마이크로 피드를 가정한 프로토타입입니다.

## 핵심 사용자 흐름

```text
앱 진입
  → 반경 안의 TMI 카드 확인
  → 좋아요 또는 넘기기
  → 댓글로 짧게 반응
  → 직접 TMI/투표 작성
  → 알림과 내 활동 확인
```

## 주요 기능

| 기능 | 설명 |
| --- | --- |
| Nearby feed | 주변 반경 안의 글만 카드로 표시 |
| Anonymous identity | 매일 바뀌는 익명 닉네임 |
| Quick reaction | 부담 없이 넘기거나 좋아요 |
| Vote card | 가벼운 양자택일 투표 |
| Comment sheet | 글의 맥락을 유지한 하단 시트 댓글 |
| Safety ready | 신고, 차단, 거리 범위 제한을 붙일 수 있는 구조 |
