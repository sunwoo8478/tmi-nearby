# Roadmap

## 1. Frontend polish

- 카드 드래그 스와이프
- 댓글 시트 애니메이션
- 모바일 실기기 safe-area 대응
- PWA 아이콘과 splash screen

## 2. Data layer

- 게시물, 댓글, 반응 mock data 분리
- localStorage 기반 임시 저장
- API 응답 타입 정리

## 3. Backend candidate

- 익명 세션 발급
- 반경 기반 게시물 조회
- 댓글/반응/투표 API
- 신고, 숨김, 차단 API

## 4. Safety

- 거리 기반 노출 제한
- 동일 사용자의 반복 신고 방지
- 민감어 필터링
- 짧은 시간 내 과도한 작성 제한
